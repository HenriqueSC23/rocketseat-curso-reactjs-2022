## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- ReactJS
- Styled components
- Zod
- React hook form
- Phosphor
- Figma

## 💻 Projeto

O Ignite Timer, como o próprio nome diz é um timer que cronometra o tempo de um determinado projeto e registra em um histórico
